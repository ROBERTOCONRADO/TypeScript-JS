Exercício

  • Crie uma interface UserData para o formulário abaixo.
  • Crie uma varíavel global UserData no window, ela será um objeto qualquer.
  • Adicione um evento ao keyup no formulário.
  • Quando o evento ocorrer adicione a {[id]: value} ao UserData
  • Salve o UserData no localStorage
  • Crie um User Type Guard, para verificar se o valor da localStorage é compatível com UserData
  • Ao refresh da página, preeencha os valores de localStorage (caso seja UserData) no formulário e em window.UserData
